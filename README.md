# Task Manager Application

A task management web application with automated deployment using CI/CD.

## Tech Stack
- Backend: Node.js / Express.js
- Database: MongoDB
- Deployment: AWS EC2
- Containerization: Docker
- CI/CD: GitHub Actions

## Features
- Create, update, and delete tasks
- RESTful APIs for task management
- Dockerized application
- Automated deployment using GitHub Actions CI/CD pipeline

## Deployment Architecture

Developer Push → GitHub Repository → GitHub Actions CI/CD → Docker Build → AWS EC2 Deployment
